# 01 Filename Ext

## Task

Напишите программу, подсчитывающую сколько раз буква встречается в предложении, а также частоту встречаемости в
процентах. 


## Пример работы программы
То есть в предложении hello world результатом будет

```
h - 1 0.1
e - 1 0.1
l - 3 0.3
o - 2 0.2
w - 1 0.1
r - 1 0.1
d - 1 0.1
```

## Сборка

`go build -o cntlet
./cntlet
`
